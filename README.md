 Izzy Task App 📝

A simple and powerful Todo App built with HTML, CSS, and JavaScript.

 🚀 Features
- Add tasks
- Delete tasks
- Mark tasks as completed
- Filter (All / Completed / Pending)
- Search tasks
- Priority levels (High, Medium, Low)
- Local storage (data is saved)

 💻 Tech Used
- HTML
- CSS
- JavaScript
